# Frontend - Tunning Manual 8080

## Описание

Фронтенд часть проекта "Tunning Manual 8080" создан с использованием нативных HTML5, CSS3 и JavaScript. Дизайн использует уникальную цветовую палитру в теплых тонах.

## Структура папок

```
front/
├── css/
│   └── style.css              # Основные стили с цветовой палитрой
├── js/
│   ├── main.js               # Скрипты главной страницы
│   ├── configurations.js     # Скрипты страницы конфигураций
│   └── hot-reload.js         # Поддержка горячей перезагрузки
├── pages/
│   └── configurations.html   # Страница конфигураций автомобилей
├── images/                   # Изображения (пока пустая)
├── index.html               # Главная страница
└── README.md               # Этот файл
```

## Цветовая палитра

Проект использует следующую палитру цветов:

- `#FFBB94` - Светло-персиковый (акцентный)
- `#FB9590` - Коралловый (вторичный)
- `#DC586D` - Розово-красный (основной)
- `#A33757` - Глубокий розовый (темный)
- `#852E4E` - Бордовый (очень темный)
- `#4C1D3D` - Темно-фиолетовый (самый темный)

## Особенности дизайна

### Главная страница (`index.html`)
- **Hero секция** с описанием проекта
- **Раздел "О проекте"** с карточками миссии, идеи и инноваций
- **Технологический стек** с описанием используемых технологий
- **Моя заинтересованность** с личным мотивационным текстом
- **Возможности системы** с 6 карточками функционала
- **Контакты** с информацией для связи

### Страница конфигураций (`pages/configurations.html`)
- **Выбор страны** с 5 карточками стран (Япония, Германия, Америка, Россия, Швеция)
- **Список автомобилей** для выбранной страны
- **Детальная информация** о каждом автомобиле
- **Характеристики** и оценка совместимости с тюнингом

## JavaScript функционал

### main.js
- Анимация появления элементов при скролле
- Плавная прокрутка к якорям
- Активная навигация при скролле
- Параллакс эффекты
- Динамическая загрузка статистики

### configurations.js
- База данных автомобилей по странам
- Интерактивные карточки стран и автомобилей
- Детальная информация о характеристиках
- Заглушки для интеграции с API

## CSS особенности

- **CSS Variables** для управления цветовой палитрой
- **CSS Grid** и **Flexbox** для адаптивной верстки
- **Backdrop-filter** для эффектов размытия
- **CSS анимации** и переходы
- **Адаптивный дизайн** для мобильных устройств

## Интеграция с бэкендом

Фронтенд готов к интеграции с REST API:

- `GET /api/cars` - получение списка автомобилей
- `GET /api/parts/car/:carId/compatible` - совместимые детали
- `POST /api/customers` - регистрация клиентов
- `POST /api/orders` - оформление заказов

## Горячая перезагрузка (Hot Reload)

Проект поддерживает горячую перезагрузку через Nodemon:

- Изменения в HTML/CSS/JS файлах автоматически применяются
- Docker контейнер следит за изменениями в папке `front/`
- Для разработки используется команда `npm run dev-docker`

## Адаптивность
