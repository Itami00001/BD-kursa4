# Tunning Manual 8080

Система подбора и оценки совместимости тюнинговых деталей для автомобилей.

## Описание проекта

Проект "Tunning Manual 8080" предназначен для подбора и оценки совместимости тюнинговых деталей для автомобилей. Каждая деталь оценивается по 10-балльной шкале совместимости с конкретной моделью автомобиля.

## Основные возможности

- **Подбор деталей:** Оценка совместимости деталей с автомобилями
- **Каталог:** Полный каталог тюнинговых деталей с фильтрацией
- **Заказы:** Оформление заказов с учетом акций
- **Услуги:** Запись на услуги установки
- **Отзывы:** Система отзывов и рейтингов
- **Акции:** Маркетинговые акции и скидки
- **Администрирование:** Управление настройками системы

## Структура базы данных

Проект использует 14 основных сущностей:

- **АВТОМОБИЛИ** - информация об автомобилях
- **КАТЕГОРИИ_ДЕТАЛЕЙ** - классификация деталей
- **ПРОИЗВОДИТЕЛИ** - производители деталей
- **ДЕТАЛИ** - каталог тюнинговых деталей
- **СОВМЕСТИМОСТЬ** - оценка совместимости
- **КЛИЕНТЫ** - информация о клиентах
- **ЗАКАЗЫ** - заказы деталей
- **АКЦИИ** - маркетинговые акции
- **ОТЗЫВЫ** - отзывы клиентов
- **УСЛУГИ_УСТАНОВКИ** - услуги по установке
- **ЗАПИСИ_НА_УСЛУГИ** - записи на установку
- **АДМИНИСТРАТОРЫ** - управление системой
- **НАСТРОЙКИ_СИСТЕМЫ** - конфигурация

## Установка и запуск

### Предварительные требования

- Docker Desktop
- Docker Compose

### Настройка окружения

1. Скопируйте файл `.env.sample` в `.env`:
```bash
cp .env.sample .env
```

2. Откройте файл `.env` и измените пароль базы данных при необходимости:
```env
DB_PASSWORD=1  # Пароль от pgAdmin
```

### Запуск проекта

1. Перейдите в директорию проекта:
```bash
cd TunningManual8080
```

2. Запустите Docker контейнеры:
```bash
docker-compose up -d --build
```

3. Откройте браузер и перейдите по адресу:
```
http://localhost:8080
```

### Доступные страницы:

- **Главная страница:** `http://localhost:8080` - описание проекта и технологического стека
- **Конфигурации:** `http://localhost:8080/pages/configurations.html` - подбор автомобилей по странам
- **API документация:** `http://localhost:8080/api-docs` - интерактивная документация API
- **API информация:** `http://localhost:8080/api` - информация о доступных эндпоинтах

### Остановка проекта

```bash
docker-compose down
```

## API Endpoints

### Категории деталей
- `GET /api/categories` - получить все категории
- `POST /api/categories` - создать новую категорию
- `GET /api/categories/:id` - получить категорию по ID
- `PUT /api/categories/:id` - обновить категорию
- `DELETE /api/categories/:id` - удалить категорию

### Детали
- `GET /api/parts` - получить все детали
- `POST /api/parts` - создать новую деталь
- `GET /api/parts/:id` - получить деталь по ID
- `PUT /api/parts/:id` - обновить деталь
- `DELETE /api/parts/:id` - удалить деталь
- `GET /api/parts/car/:carId/compatible` - получить совместимые детали для автомобиля

### Автомобили
- `GET /api/cars` - получить все автомобили
- `POST /api/cars` - создать новый автомобиль
- `GET /api/cars/:id` - получить автомобиль по ID
- `PUT /api/cars/:id` - обновить автомобиль
- `DELETE /api/cars/:id` - удалить автомобиль
