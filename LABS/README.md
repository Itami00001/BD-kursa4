# Лабораторные работы по курсу "Базы данных"

## Структура лабораторных работ

### ЛР 8-10: Базовые CRUD операции
- **ЛР 8:** Создание и чтение записей
- **ЛР 9:** Обновление и удаление записей  
- **ЛР 10:** Комплексные CRUD операции с валидацией

### ЛР 11-12: Транзакции и блокировки
- **ЛР 11:** Основы транзакций и уровни изоляции
- **ЛР 12:** Пессимистические и оптимистичные блокировки

### ЛР 13-14: Продвинутые запросы и оптимизация
- **ЛР 13:** Сложные SQL запросы и агрегации
- **ЛР 14:** Оптимизация производительности и индексация

## Запуск лабораторных работ

Каждая лабораторная работа содержит:
- Теоретическую часть
- Практические задания
- Примеры кода
- Тестовые данные
